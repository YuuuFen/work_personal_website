# 網站功能規劃

**前台：**

1. 導覽列
2. 個人介紹
3. 作品集區塊，作品以提供連結跳轉
4. 頁尾區塊，可導向後台登入畫面

**後台：**

1. 登入功能
2. 後台功能
   - 個人基本資料管理
     - 自我介紹
     - 興趣
     - 技能
   - 作品管理
     - 可新增作品項目
     - 可分類作品
     - 可上傳作品縮圖
     - 作品列表
     - 作品項目的更新/刪除
   - 網站管理
     - 配色管理（主顏色、強調色、背景色）
     - 可透過設定改變網站部份視覺
     - 可設定網站各區塊或項目的顯示與否

**前台未實現功能：**

1. RWD 功能
2. 可切換淺/深色模式
3. 可切換顯示語言
